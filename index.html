INTRODUCTION

HTML is the standard markup language for creating Web pages, it stands for Hyper Text Markup Language.
    HTML describes the structure of a Web page and consists of a series of elements. HTML elements tell the browser how to display the content.
    HTML elements are represented by tags. Tags label pieces of content such as "heading", "paragraph", "table", and so on.
    Browsers do not display the HTML tags, but use them to render the content of the page.
